# Conky Desktop Guru
## - Install
   - Run the following command to install conky:

         sudo apt install conky-all
   - To create the config file run:
   
         conky -C > ~/Documents/conky.config
   - Create the conky folder in the home directory:
  
         mkdir ~/.config/conky
